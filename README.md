# Program-to-Find-the-Maximum-Value-in-a-List-Using-a-Loop-Without-Using-max-
This Python program finds the maximum (largest) value in a list using a loop, without using the built-in max() function.  The program begins with a predefined list of integers. It assumes that the first element of the list is the largest value. Then, it iterates through each element in the list using a for loop.
